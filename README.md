# OpenClaw 智能助理 🦀

OpenClaw 是一款开源的智能大模型 Agent（智能体）框架。我们刚刚为它深度定制了**出行与生活实况能力**，使它不仅仅是一个聊天机器，更是一位懂出行的“生活管家”。

## ✨ 核心特色功能

1. **图文并茂的“聪明出行”向导 (`smart_trip`)**
   - 🗺️ **导航地图生成**：底层接入高德地图 API，自动规划并实时生成直观的含车辆、步行及目的地停车场的微缩路线图。
   - 📸 **网友实拍聚合**：独创集成小红书相关场景搜索，把景点的最新“网友实拍”和避坑指南，像朋友圈一样直接一排 3 张图展示在聊天窗口中，不仅有文字干货也有现场美图。
   - 📝 **智能排版系统**：生成的各种卡片和缩略地图都采用精致的 Markdown 排版，完美兼容各类端内渲染。
   - 🧠 **决策辅助**：自动结合新闻、天气及周边路况对出行时间（如推荐错峰出门）给出贴心建议。

2. **多模态与超长上下文基础**
   - 核心支持调用如 `Doubao-Seed-2.0-lite` 这一类具备高保真、多源信息融合的新一代大语言模型。
   - 长期记忆（`MEMORY`）与偏好保存：您的习惯一旦被记录，永久生效。

## 🚀 如何开始使用

### 1. 填入你自己的密钥 (API Keys)

> ⚠️ **安全警告**：本项目出于安全考虑，**没有内置或上传任何第三方服务的 API Key**。要使用搜索、地图等工具，你必须配置自己的访问凭证！

请在您部署的运行环境（通常是 `.env` 或相关的系统环境变量）里填入以下必需的密钥：

- **`AMAP_API_KEY`**：高德地图 Web 服务 API Key。用于路线规划、地理编码与静态地图生成。你可以前往 [高德开放平台](https://lbs.amap.com/) 免费申请。
- **`GEMINI_API_KEY`** (或其他您使用的大模型 Key)：作为底座模型或进行外部内容处理。
- `XHS` 及其他服务的配置请按系统内部要求在启动设置中调整。

### 2. 启动服务

进入项目根目录后，确保已安装 Node.js 及必要环境，运行：

```bash
# 启动 Gateway 后端服务
node scripts/run-node.mjs gateway

# 然后在前端按照常规流程访问对话 UI 即可
```

### 3. 可以像这样考考它：

- _"今晚下班想去西溪湿地看灯会，现在那边什么情况？"_
- _"帮我看看怎么去西湖银泰性价比最高？最好有网友最近的实拍反馈。"_

## 👨‍💻 技术排版说明 (给开发者的提示)

我们对输出做了精细控制，在服务端使用 `sharp` 对原始网络地图和实况图片进行了尺寸重塑（地图限制在 350px 宽，组图限制在 180px 宽并采用 Markdown 空格内联法），这意味着它可以丝滑呈现在任何支持 Markdown 的小屏或分屏聊天组件内，而不会把版面撑爆。
